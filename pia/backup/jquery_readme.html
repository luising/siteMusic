Instrucciónes de jQuery Plugin

jQuery es una increíble colección de JavaScript que le ayuda a crear los efectos web maravillosos por medio de sólo unas pocas líneas de código.
Los plugines jQuery le ayudan a mejorar la interfaz de su sitio web y distinguirlo de otros sitios.
Los efectos jQuery comunes son menús desplegables, animaciones y otros elementos.
